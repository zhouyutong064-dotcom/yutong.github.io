      使用说明

      快速上手指南

      1

      上传报告

      从 Coupang 广告后台下载广告报告（xlsx 格式），拖拽或点击上传

      2

      查看分析

      系统自动计算 CTR/CVR/CPC/ROAS/ACoS，并对关键词进行高效/低效/无效/无意义分类

      3

      非强相关词筛查

      查看非强相关词筛查，一键添加否词，精准清理无效流量

      4

      AI 优化建议（可选）

      配置 DeepSeek 或豆包 API Key，获取广告活动维度的专业优化方案，AI建议否词自动同步到否词清单

      5

      导出结果

      一键导出包含分析结果的 Excel，直接复制否词清单到广告后台
